This directory includes all the code used for creating and training both the MAE-ViT model, and the fine-tuned model.
MAE-ViT model is created in the Main.py file, while the fine-tuned model is created and traind in the fine_tuning.py file.
The rest of the files are supportive files, with classes, methods and functions to facilitate the everything in main.py and fune_tuning-py
